# Proyecto Final - Sistema de Gestión de Estudiantes

**Descripción**: Desarrollar una aplicación de consola en C# que permita administrar información
básica de estudiantes utilizando los conocimientos adquiridos durante el curso.

**Objetivo General**: Aplicar los conceptos fundamentales de programación en C# mediante el
desarrollo de un sistema que permita registrar, consultar y gestionar estudiantes.

**Funcionalidades mínimas**:
1. Registrar estudiante (Matrícula, Nombre, Edad y Nota Final). 
2. Mostrar todos los estudiantes registrados. 
3. Buscar estudiante por matrícula.
4. Calcular promedio general del curso. 
5. Mostrar estudiante con mayor nota. 
6. Mostrar estudiantes aprobados (nota ≥ 70). 
7. Mostrar estudiantes reprobados (nota < 70). 
8. Salir del sistema. 


**Menú Principal**:
1. Registrar estudiante 
2. Mostrar estudiantes 
3. Buscar estudiante 
4. Calcular promedio 
5. Mayor nota 
6. Estudiantes aprobados 
7. Estudiantes reprobados 
8. Salir 


**Temas Aplicados**:

- Variables y constantes
- Operadores
- Condicionales
- Estructuras repetitivas
- Arreglos y matrices
- Variables de referencia básicas
- Funciones


**Requisitos Técnicos**:

- Utilizar al menos 5 funciones.
- Utilizar al menos un ciclo while o do while.
- Utilizar al menos un ciclo for o foreach. 
- Utilizar estructuras if. 
- Utilizar arreglos para almacenar información. 

**Valor Extra (Opcional)**:
- Eliminar estudiante. 
- Modificar nota. 
- Mostrar cantidad de estudiantes registrados. 
- Validar entradas con try-catch. 
   
**Criterios de Evaluación**:

- Funcionamiento del menú (20 pts)
- Registro de estudiantes (20 pts)
- Uso correcto de funciones (20 pts)
- Uso de ciclos y condicionales (15 pts)Uso de arreglos (15 pts)
- Presentación y organización (10 pts)
- Total: 100 puntos
